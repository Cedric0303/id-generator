# cedric0303/id-generator

## Installation

```bash
# npm
npm install --global @cedric0303/id-generator

# yarn
yarn global add @cedric0303/id-generator

# bun
bun add --global @cedric0303/id-generator
```

## Usage

```bash
> sgid
A5758052

> hkid
S8541816E

// output automatically copied to clipboard
```
